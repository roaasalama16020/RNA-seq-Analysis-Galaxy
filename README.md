

# RNA-seq Analysis Using Galaxy
## Overview
This repository contains an academic team project completed as part of the Bioinformatics course at Misr University for Science and Technology (MUST).
The project demonstrates a complete RNA-seq analysis workflow using the Galaxy platform, starting from raw sequencing data and ending with differential gene expression and functional enrichment analysis.
---
## Project Type
**Academic Team Project**
---
## My Contribution
I was responsible for the **gene quantification (counting)** stage of the RNA-seq analysis workflow using the Galaxy platform.
My contributions included:
- Performed gene quantification using **featureCounts**.
- Generated the read count matrix required for downstream differential gene expression analysis.
- Verified the counting results before proceeding to the next stage of the workflow.
Through this work, I gained hands-on experience with RNA-seq gene quantification and bioinformatics workflows using the Galaxy platform.

## Workflow
1. Data collection from NCBI BioProject and SRA.
2. Quality control of raw sequencing reads.
3. Read alignment using HISAT2.
4. BAM file processing using SAMtools.
5. Quality assessment using MultiQC.
6. Gene quantification using featureCounts.
7. Differential gene expression analysis using DESeq2.
8. Functional enrichment analysis.
